Module with different serializers that can be used with this crate.

By default, a number of serializers are supplied that can be used by turning on the feature flag.

FeatureDescription
bincode-serializationserialisation using bincode (enabled by default) .
rmp-serializationserialisation using rmp-serde .

Modules

bincode

Implementation of SerialisationStrategy for serializing with bincode. It is enabled with the bincode-serialization feature flag.

rmp

Implementation of SerialisationStrategy for serializing with rmp-serde. It is enabled with the rmp-serialization feature flag.
